# PrayerTime
Script used to calculate prayer timings as per website "https://www.sunrise-and-sunset.com"
User can enter Year in format YYYY e.g 2024 and then prayer data could be processed by copying tables from the given website
The calculated times are then saved in the excel sheet [CSV format] with name like "PrayerTiming2024.csv"
Note: run the script in command prompt with command > python GetPrayerTime_v01.py

# RamzanPrayerTime
Script used to calculate prayer timings as per website "https://www.sunrise-and-sunset.com"
User can enter Year in format YYYY e.g 2024 , Start Month e.g. 1 for January, Start Date (1 to 31) and End Data (1 to 31, without End month)
and then ramzan prayer data could be processed by copying tables from the given website
The calculated times are then saved in the excel sheet [CSV format] with name like "RamzanTiming2024.csv"
Note: run the script in command prompt with command > python GetRamzanPrayerTime_v01.py
